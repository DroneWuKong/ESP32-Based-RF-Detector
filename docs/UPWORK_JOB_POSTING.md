JOB DESCRIPTION:
═══════════════════════════════════════════════════════════════════════════

I need a simple 2-layer PCB designed for an RF detector project. I have COMPLETE specifications including:
- Full schematic with all connections
- Component list with exact part numbers
- Component placement coordinates
- Trace width requirements
- Design rules

This is a straightforward power management board - no complex RF routing.

BOARD SPECIFICATIONS:
- Size: 40mm x 40mm
- Layers: 2 (Top signal, Bottom GND plane)
- Corner radius: 2mm
- Material: FR4 1.6mm
- Components: ~15 parts (regulator, capacitors, connectors, diode, fuse)

YOUR TASK:
1. Review my complete specifications (provided in attached files)
2. Create schematic in KiCad/Altium
3. Create PCB layout with component placement at specified coordinates
4. Route all traces according to width specifications
5. Add GND plane on bottom layer with thermal reliefs
6. Run DRC (Design Rule Check) and fix violations
7. Generate Gerber files ready for JLCPCB manufacturing
8. Generate drill files (Excellon format)
9. Provide BOM with LCSC part numbers (for JLCPCB assembly if desired)

WHAT I'M PROVIDING:
✅ Complete component list with values and packages
✅ Full netlist (every connection specified)
✅ Component placement coordinates (x, y positions)
✅ Trace width requirements per net
✅ Design rules (clearances, via sizes, etc.)
✅ Board outline specifications
✅ Reference documentation

DELIVERABLES:
□ KiCad project files (.kicad_pro, .kicad_sch, .kicad_pcb) OR Altium files
□ Gerber files (RS-274X format) for JLCPCB:
  - F.Cu (Top Copper)
  - B.Cu (Bottom Copper)
  - F.Mask (Top Solder Mask)
  - B.Mask (Bottom Solder Mask)
  - F.SilkS (Top Silkscreen)
  - B.SilkS (Bottom Silkscreen)
  - Edge.Cuts (Board Outline)
□ Drill file (Excellon format)
□ BOM (Bill of Materials) CSV with LCSC part numbers
□ Assembly drawing (component placement diagram)
□ DRC report (showing zero errors)

REQUIREMENTS:
- Experience with KiCad 7+ or Altium Designer
- Experience with JLCPCB manufacturing requirements
- Good communication in English
- Ability to follow detailed specifications
- Available to start immediately

TIMELINE:
- 3-5 days from project start
- Milestone 1 (50%): Schematic complete + initial PCB layout
- Milestone 2 (50%): Final Gerbers + all deliverables

BUDGET:
$75 USD (fixed price)
- $37.50 on schematic completion + layout review

BONUS (+$25):
